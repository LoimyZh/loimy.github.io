加前缀可以让动词的意义具体化

1. Б

  1. -бивать // -бить (бью, бьёшь) 敲，打，凿；打碎

    • забивать // забить ①钉，砸；②（球类的）踢，打，得分
      забивать гвозди в стену - 把钉子钉入墙壁
      забить гол - 进球
    • пробивать // пробить 穿透，打通；（光线等的）穿过，透过
      пробить стену - 把墙壁凿通
      лучи солнца пробили окно - 太阳光透过了窗户
    • вбивать // вбить 钉，打进
      вбить гвоздь в стену - 把钉子钉到墙上去
    • перебивать // перебить ①打死（很多），打碎（很多）；②打断话
      перебить стаканы - 打碎很多杯子
      перебить чьи слова - 打断某人发言
    • убивать // убить 杀死，杀害；扼杀
      убить самого себя - 自杀
      убить талант - 扼杀天分
    • разбивать // разбить 打破，打碎，使……破灭
      разбить мечту - 使梦想破灭
    • отбивать // отбить ①打碎了某个事物的一个部分；②（用武力）夺回
      отбить ручку чайника - 打碎了茶杯把手
      отбить город - 夺回城市
    • сбивать // сбить ①打下来（树上的苹果，天上的鸟、飞机）
      сбить вражеский самолёт - 击落敌机
    • 真题：В последний момент игры наша команда _____ решающий гол и победила. (2005-63)
      A) пробила
      B) забила забить гол - 进球，固定搭配
      C) вбила
      D) перебила

  2. -бирать // -брать 拿；买；抓；借；贷

    • забирать // забрать 拿走，带走；拘捕
      забрать с собой все книги - 拿走所有书籍
    • подбирать // подобрать 拾起，捡起；(数量少地)采摘；挑选，选配（选择合适的人/物）
      подобрать рассыпанные вещи - 捡起散落的东西
      подбирать ключ к замку - 给锁配上钥匙
    • набирать // набрать (大量地)采摘；招聘，招募；加足, 达到(某种限度)；拨(号)
      набрать корзину грибов - 采一筐蘑菇
      набирать рабочих на завод - 工厂招收工人
      набирать скорость - 增加速度
      набирать высоту - 爬高; 上升
      набирать номер телефона - 拨号码
    • собирать // собрать 集合，收集；装配，组装
      собрать студентов - 把学生召集到一起
      собирать марки - 收藏邮票
      собрать машину - 把机器装配好
    • убирать // убрать 收拾，拿走，拿开；收割, 收获
      убирать со стола - 收拾桌子
      убрать книги с полки - 把书从书架上拿走
      убирать урожай - 收割庄稼; 收获
    • избирать // избрать 选举
    • 其被动形动词 избранный - 卓越的*
      избрать кого-либо депутатом(чем) \ в депутаты(в+形同复一的复四) - 选…为代表
      избирать кого-либо в президиум - 把…选入主席团
    • перебирать // перебрать 逐一查看，翻阅；逐一想起
      перебирать старые письма - 翻阅旧的信
      перебирать в памяти - 逐一想起
    • выбирать // выбрать 选择（较好的，所需要的）
    • выбор - 选择；выборы - 选举*
      выбрать профессию - 选定职业
    • отбирать // отобрать 选拔出；收去 ，收回
      отобрать самые спелые яблоки - 挑出最熟的苹果
      отбирать входные билеты - 收入场券
    • разбирать // разобрать (很多人把所有的东西)分别拿走；拆开（反义词собрать）
      Пришли рабочие, разобрали лопаты и начали работать. 工人们来了, 各人拿起铲子, 就干起活来了。
      разобрать машину - 把机器卸开

  3. более

    • тем более - ①况且，更何况(=более того)；②（连接词）тем более, что - 尤其[是]; 何况；③（语气词）尤其
      Останемся дома, тем более собирается гроза - 我们留在家里, 何况眼看就要来暴风雨
      Климат здесь не здоровый, тем более для ребёнка. - 这儿的气候不好，尤其对于孩子（语气词）
      和平发展非常重要，尤其是对于发展中国家。 - Мирное развитие очень важно, тем более для развивающихся стран
    • более того - 况且，更何况，不仅如此
      Ваша деятельность бесполезна. Более того(тем более), она даже вредна - 你的行为是没有益处的，更何况他是有害的
    • тем не менее - 就算是……，还；尽管如此
      Ему 70 лет, тем не менее он физически очень крепок. - 他70岁了，身体却硬朗得很。
    • (и) тем самым - 这样；于是
    • более или менее - 或多或少
    • мало того - 况且; 此外
    • куда(гораздо/ намного) более - 非常多的
    • более чем - 多于（副词短语，没有接格）
      С помощью этой системы можно перевести деньги в более чем（副词短语，即使把他去掉也是接во что） 30 стран мира, включая Китай, Израиль и Бразилию.
      За последние годы зарплата в России поднялась более чем на 20%.

倍数表达法：во сколько раз / на сколько процентов / до скольких процентов
Увеличиться в четыре раза. - 增加了三倍/增加到四倍/是原来的四倍

  • менее чем - 少于
  • во что бы то ни стало - （连接词）不论如何，不管怎样
  • 真题：Ну как же вы посмели（敢） так дурно（不好） думать о людях и ____ об учителе? (2003-72)
  • A) тем более* 你怎么敢这么不好地想别人，更何况是你的老师呢
    B) кроме того
    C) тем не менее
    D) тем самым
    Мы очень торопились（торопиться - 忙于，急于）, _____ мы не смогли все закончить вовремя. (2005-71)
    A) более или менее
    B) более того
    C) тем не менее 就算是这样，还是没及时赶上
    D) тем более

2. В

  1. вид

    • на вид = с виду = по виду - 外表上看来
    • иметь виды на кого-что - 打某人的主意；指望什么东西
    • иметь в виду - 说的是…, 指的是…
    • под видом кого - 冒充某人
    • под видом чего - 以……为借口
    • в сокращённом виде - （作品）节选；经删减的，不完整版
    • из виду кого - 在视野之外
    • в виду кого - 在视野之内
    • делать вид - 装样子，做样子
    • при виде кого-чего - 看到某人、某物时
    • на виду у кого - 当着某人的面
    • 真题：Когда мы говорим глаголы движения, мы имеем ____ такие слова, как ходить, идти и т.д. （2004-60）
      A) на вид
      B) виды
      C) в виду 我们指的是
      D) под видом

  2. -влекать // -влечь(-еку, -ечёшь, -екут)

    • увлекать // увлечь (у- 表示的是行为的彻底性)
      ①吸引，使……着迷；②带去，引去；(водой)冲走；(ветром) 刮走
      интересная работа увлекла его - 有兴趣的工作使他全神贯注了
      толпа увлекла детей на площадь - 人群把孩子们引到广场去了
    • привлекать // привлечь 吸引来，吸引参加，吸引到谁身上(к кому-чему)
      привлечь его к работе в комиссии - 吸收他来参加委员会的工作
      привлекать внимание - 吸引注意力
    • извлекать // извлечь (кого-что из кого-чего)从… 提取；吸取
      извлекать сок из растения - 从植物中提取汁液
      извлечь урок - 吸取教训
    • отвлекать // отвлечь (кого-что от чего)引诱…离开，使…放下，转移(注意力)
      отвлекать внимание - 转移注意力
      отвлекать кого-либо от работы - 使…放(丢)下工作
    • 真题：Все знают, что Лобанов умеет ____ к себе своим умом. （2004-68）
      A) увлекать
      B) привлекать 用自己的智慧(чем)把注意力吸引到自己的身上(к себе)
      C) извлекать
      D) отвлекать

  3. внимание

    • кто-что привлекает внимание - 谁吸引注意力
    • кто уделяет внимание кому-чему - 某人关注什么
    • кто обращает внимание на кого-что - 某人关注什么
    • 真题：В последнее время все большее внимание западных архитекторов _____ Китай. (2005-64)
      A) увлекает
      B) привлекает 中国吸引西方建筑家的注意力
      C) уделяет
      D) обращает
  1. -вёртываться // -вернуться (-нусь, -нёшься)
    • развёртываться // развернуться ①把卷着的东西打开；②(得到)展开; (才能、专长等)得以施展
      Ковёр развернулся. - 地毯铺开了。
      Талант развернулся. - 才能得到发挥了。
    • свёртываться // свернуться ①把东西卷起来；②(牛奶)变成乳渣状；③缩减; (活动, 业务)暂时减少(或停止)
      Кошка свернулась клубком. - 猫卷缩成团了。
      Молоко свернулось. - 牛奶凝成块了
      Производство свернулось. - 生产缩减了
    • завёртываться // завернуться 把自己裹起来；卷起来
      завёртываться в одеяло - 围上被子(во что)
      рукав завернулся - 袖子撸起来
    • повёртываться // повернуться (чем куда) 转；转身
      повернуться на другой бок - 翻身
      повернуться лицом к кому-чему-либо - 向…转过脸去
    • повёртывать // повернуть (что) 转（什么部位）
      повернуть голову 转头
    • 真题：Вася ____ на другой бок и взглянул на（看一看） будильник: пора вставать. (2003-73)
      A) развернулся
      B) cвернулся
      C) завернулся

D) повернулся 翻身，看一眼闹钟：该起了

3. Г

  1. глаз 眼睛
    • в моих глазах 依我看 = на мой взгляд
      Вы просто герой в моих глазах. - 你在我眼中简直是英雄。
    • на глазах 当着谁的面（у кого）；眼看着，很快
      Прямо у нас на глазах руку на женщину поднял. - 当着我们的面就动手打女人。

поднять руку на кого - 动手打某人
Город растёт на глазах. - 城市眼看着就发展起来了。

  • перед глазами 在…的眼前（物理意义上的眼前）
    У меня нет текта перед глазами. - 我手边没有这篇文章。
  • на глаз 大约，用眼估计 = с виду
    Я определяю на глаз. - 我用眼估计
  • с глазу на глаз 面对面地，两人单独地
  • говорить в глаза 当面说
  • за глаза 背地里
    ни в глаза, ни за глазами - 无论当面, 还是背后
  • У страха глаза велики. - 草木皆兵（眼睛瞪得非常大）

    • опустить голову - 低头

    • голова в доме - 一家之主
    • без головы - 不假思索

4. Д

  1. -давать // -дать

    • выдавать // выдать ①（按照一定的程序，规则，自上而下）发给（很多人）；②泄漏；出卖
      выдавать зарплату - 发给工资
      выдать военную тайну - 泄漏军事秘密
      выдать друга - 出卖朋友
    • отдавать // отдать ①归还；②贡献；献给(кому-чему/за что)
      отдавать долг - 还债
      отдать жизнь за свободу Родины - 为祖国的自由而献出生命
      отдать себе отчет - 反思，反省
    • передавать // передать 交给，转交；传递；转达；传输
      передать привет - 转达问候
    • подавать // подать ①递[给]；端[给]；②上菜；③递交
      подать кому-либо стакан воды - 拿一杯水端给
      сначала подайте суп - 先端上汤来吧
      подавать заявление - 递申请书; 提出申请书
    • задавать // задать ①指定，规定；②提问；③举行，举办
      задать срок - 规定期限
      задать кому вопрос - 提问
      задавать пир - 设宴
    • издавать // издать ①出版，发行；②颁布
      издать книгу - 出版书
      издатель - 出版者； издательство - 出版社
      издавать приказ - 颁布命令
    • придавать // придать 赋予
      придавать документу законную силу - 赋予文件以法律效益
      придать кому-чему большое значение - 赋予……以重大意义
    • продавать // продать ①卖，出售；②出卖，叛卖
      продать своих друзей - 出卖朋友
    • раздавать // раздать （一一地）分配，分派
      раздать роли актёрам 给演员们分配角色
    • сдавать // сдать ①(нсв.)参加考试；(сов.)通过考试；②租出去（自己租 - снимать // снять）；③移交，交[给]
      сдавать комнату - 租出一个房间
      сдавать дела - 移交(交代)工作
    • 真题：Всем студентам, успешно закончившим обучение в университете, в торжественной обстановке（在隆重的气氛下） ______ дипломы. （2004-64）
      A) отдали 发毕业证
      B) выдали

      D) подали

  2. -даваться // -даться

    • удаваться // удаться ①顺利完成，成功；②[无人称](кому 接完成体不定式)办到, 做成
      Попытка не удалась. - 企图没有成功。
      Мне не удалось поехать. - 我没有去成。
    • передаваться // передаться （疾病）传播 + кому-чему
      Болезнь передалась ребёнку. - 病传给了婴儿。
    • выдаваться // выдаться ①(不用一, 二人称)突出，伸出（物理意义上）；②出众，杰出 (чем)；③(不用一, 二人称)<口>(现象，机会)出现, 遇到; 发生(常指季节, 天气等)
      Дом выдался углом на площадь. - 房子的一角向广场探出

      выдаться хороший денёк. - 遇到了一个好天儿
      выдаться случай - 遇到了机会
    • задаваться // задаться ①〈口〉对自己提出(问题、任务、目的等)(+чем)；②〈口〉(某种日子、天气)出现
      задаться вопросом - 对自己提出问题(准备解决)
      Хороший денёк задался. - 遇着了好天气。
    • 真题：Я навещаю（拜访某人） своих родственников, как только ____ свободный день. (2003-61)

      B) передается
